# Develop A Habit

Telegram-бот для трекинга привычек, дневника и статистики.

## Запуск локально

1. Создайте и активируйте виртуальное окружение.
2. Установите зависимости:
   ```bash
   pip install -e .[dev]
   ```
3. Скопируйте `.env.example` в `.env` и заполните значения.
4. Примените миграции Alembic.
5. Запустите бота:
   ```bash
   python -m develop_a_habit.main
   ```

## Структура

- `src/develop_a_habit` — исходный код приложения.
- `migrations` — миграции БД.
- `docs` — спецификации и журнал коммитов.
- `tests` — тесты.
- `scripts` — служебные скрипты.
